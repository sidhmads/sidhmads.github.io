NUSpace
In order to learn more about System on Chip (SoC) and assembly language an imaginary project was given.

NUSpace is a manned space flight system programmed on LPC1769 which runs on a ARM Cortex-M3 microcontroller.This system sends data periodically to a server known as NUSCloud. NUSpace has three modes of operation namely, Stationary, Launch and Return. If certain conditions are met, it will transmit to NUSCloud. The stationary mode is when the rocket is still in its launchpad. Lauch mode is when the rocket is launched towards space. Return mode is when the space shuttle mounted on the rocket is detached and returns to land on earth.

The whole NUSpace system was designed from scratch to obtain hands on experience.
Tech Stack: ARM Cortex-M3 microcontroller, Assembly & C Programming
GitHub Repository
